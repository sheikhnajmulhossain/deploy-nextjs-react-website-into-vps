# Building the project 

1. navigate to your repo file in vps
`cd filename`   

2. Install bun/npm (the package manager your using)
`npm install -g bun`
`bun install`
or
`npm install`

3. Run 
a. for one time run 
`bun dev`
or for npm
`npm start`

b. for continues run in the vps background 
`bun dev &`
or for npm
`npm start &`
